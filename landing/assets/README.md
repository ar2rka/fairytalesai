# Assets Directory

## Logo Files Required

Place your logo files in this directory:

1. **logo.png** - Main logo file (recommended: 300x300px or larger, square format)
   - Used in hero section and footer
   - Should be high quality with transparent background if possible

2. **favicon.png** - Favicon file (recommended: 32x32px or 64x64px)
   - Used as browser tab icon
   - Square format, will be automatically resized by browsers

## File Naming

- Main logo: `logo.png`
- Favicon: `favicon.png`

## Image Format

- PNG format recommended (supports transparency)
- SVG format also supported (update file extensions in HTML if using SVG)

## Creating Favicon from Logo

You can create the favicon from your logo using:
- Online tools: https://favicon.io/favicon-converter/
- Image editing software: Resize logo to 32x32px or 64x64px
- Command line: Use ImageMagick or similar tools

## Notes

- The logo will be automatically sized by CSS
- Hero logo: 120px (mobile) / 150px (desktop)
- Footer logo: 60px
- Favicon: Browser will resize automatically
